Git là gì

- Git là một mạng xã hội dành cho Dev

- Git là một hệ thống điều khiển phiên bản( version control system) theo hình thức phân tán

- Git được sử dụng để quản lý mã nguồn (source code) và ghi nhận các thay đổi

Câu lệnh git

- git init : khởi tạo local repository

- git add “tên file” : đưa các file vào vùng nhớ staging area

- git commit -m ”ghi chú” : đẩy các file trong staging area vào local repository

- git remote add origin https://github.com/”tên tk”/GIT.git : kết nối local repository với remote repository

- git push origin master : đẩy code từ local repo lên remote repository
